Cara Instalasi jika download melalui Google Drive

1. Buka terminal
2. membuat environment "python -m venv [NAMA_ENV]"
3. aktifkan environment
	- windows "[NAMA_ENV]\Scripts\activate.bat" atau "[NAMA_ENV]\Scripts\Activate.ps1"
	- mac source "[NAMA_ENV]/bin/activate"

4. install requirements.txt "pip install -r requirements.txt"
Cara Penggunaan

cukup menggunakan perintah pada terminal "python app.py" lalu mengisi input yang dibutuhkan oleh sistem
